# St. Louis, Missouri, USA Storm Detections - May 2025

BirdWeather PUC detections in the St. Louis metro area surrounding the
May 15-16, 2025 tornado outbreak, including an EF3 tornado.

## Usage

``` r
stl_storm_May2025
```

## Format

A data.table with detection and station columns as returned by
[`get_detections()`](https://brentpease1.github.io/birdweatheR/reference/get_detections.md)

## Source

BirdWeather API via birdweatheR
