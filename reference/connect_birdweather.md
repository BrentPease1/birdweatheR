# Connect to the BirdWeather API

Establishes a connection to the BirdWeather GraphQL API. Must be called
once per session before using any other package functions.

## Usage

``` r
connect_birdweather()
```

## Value

Invisibly returns the connection object

## Examples

``` r
if (FALSE) { # \dontrun{
connect_birdweather()
} # }
```
